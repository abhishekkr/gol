Golang utility package bundle

,
any common utility required while developing go projects comes here to evolve
  • golgoquery - easy web scraping using goquery at base
  • gollog - easy integration for better logging
  • golrandom - for random values
  • golregex - to use regex common usecases
  • golnats - to easily use NATS messaging queue
  • golenv - access and check on environment variables
Reach out to us on developer channel for more details.
