Golang utility library bundle

extensiongolnats

This package helps with using NATS messaging queue easily. Currently it provides:

  • provides a Nats struct bundling use of queue featurescode
  • allows to use Request and Reply using NATScode
  • allows to use Publish and Subscribe modelcode
  • Subsribers could be used in Sync and Async modecode
it plans to include more customization on using NATS features

Reach out to us on developer channel for more details.
